# חיזוי עזיבת עובדים | Employee Churn Prediction

פרויקט מסכם בלמידת מכונה שמטרתו לחזות האם עובד צפוי לעזוב את הארגון, ולזהות את הגורמים
המשפיעים ביותר על העזיבה — תובנה בעלת ערך עסקי ישיר עבור מחלקת משאבי אנוש.

## תוכן הפרויקט

| קובץ | תיאור |
|------|-------|
| `Employee_Churn_Prediction.ipynb` | המחברת המרכזית — כל שלבי הפרויקט מקצה לקצה |
| `employee_churn.csv` | מערך הנתונים (נוצר אוטומטית בעת הרצת המחברת) |
| `requirements.txt` | רשימת הספריות הנדרשות |

## מתודולוגיה

1. **שלב 1–2** — טעינת הנתונים, הצגה וסקירה סטטיסטית.
2. **שלב 3** — הדמיה גרפית (Seaborn) של הגורמים המשפיעים על עזיבה.
3. **שלב 4** — ניקוי הנתונים וקידוד (Ordinal ו-One-Hot Encoding).
4. **שלב 5** — חלוקה ל-Train/Test ואימון מודלים (Logistic Regression ו-Random Forest).
5. **שלב 6–7** — הערכת המודל (Accuracy, Confusion Matrix, ROC/AUC), חשיבות המאפיינים ומסקנות.

## הפעלה

```bash
pip install -r requirements.txt
jupyter lab Employee_Churn_Prediction.ipynb
```

לאחר פתיחת המחברת, יש להריץ את התאים לפי הסדר (Run All). מערך הנתונים נוצר בתוך המחברת עצמה,
כך שהפרויקט **עצמאי לחלוטין** ואינו דורש חיבור חיצוני.

## ספריות בשימוש

Pandas, NumPy, Matplotlib/Seaborn, Scikit-learn בלבד — בהתאם לדרישות הפרויקט.

## תוצאות עיקריות

- דיוק (Accuracy) של כ-83% בקבוצת הבדיקה, יציב תחת Cross-Validation.
- AUC של כ-0.89.
- הגורם המשפיע ביותר על עזיבה: **רמת שביעות הרצון** של העובד.
